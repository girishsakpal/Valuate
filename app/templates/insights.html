Pipeline

what happens between input and prediction
  1. Stratified 80/20 split on binned median_income to keep train/test representative
  2. Numeric columns → median imputation → standard scaling
  3. Categorical ocean_proximity → one-hot encoding (unseen categories ignored)
  4. Candidates compared via 10-fold CV: Linear Regression, Decision Tree, Random Forest
  5. Winning model refit on full training set, scored once on held-out test set
  6. Pipeline + model persisted with joblib, served through Flask for live inference
